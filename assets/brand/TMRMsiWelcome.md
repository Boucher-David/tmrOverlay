# Tech Mates Racing Overlay

Install the overlay app, update-ready package identity, and shortcuts for launch from Windows.

This installer removes the application binaries and shortcuts during uninstall. TmrOverlay also removes its local app-data root during uninstall, including settings, history, logs, diagnostics, captures, runtime state, imported Garage Cover images, and user-generated track maps stored under `%LOCALAPPDATA%\TmrOverlay`.
