# Overlay Catalog Mocks

Static design previews for RaceLab/iOverlay-style overlay kinds rendered with current TmrOverlay visual tokens.

These are exploratory mockups, not runtime screenshots. They cover the public overlay concepts from RaceLab's overlay image catalog plus iOverlay-only concepts such as Twitch chat and garage/setup cover.

Regenerate with:

```bash
swift tools/render_overlay_catalog.swift
```

Main review sheet:

```text
mocks/overlay-catalog/overlay-catalog.png
```
