# Fuel Calculator Mocks

Open `fuel-calculator-states.png` for a shareable preview, or `fuel-calculator-states.svg` for the editable source image.

Included states:

- Waiting for usable fuel data
- Live green-flag strategy
- User-history-informed strategy
- Long-race rolling stint window
- No-stop compact state
- Spotting/model fallback state

The SVG is intentionally static. It is meant for feedback on content density, wording, and layout, not as a runtime screenshot.
